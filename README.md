## Команды бота

| Команда                           | Описание                                           |
|-----------------------------------|----------------------------------------------------|
| `/xkcd`                           | случайная картина из [xkcd.com](https://xkcd.com/) |
| `/today`, `/tomorrow`, `/lessons` | узнать расписание на `сегодня`, `завтра`, `неделю` |
| `/dick`, `/top_dick`              | игра: по выращиванию своего хозяйства              |
| `/gay`, `/top_gay`                | игра: узнать кто gay дня                           |
